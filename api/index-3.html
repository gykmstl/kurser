API - BalderJS 3.0

Canvas properties

W: number
H: number

Width och height of the canvas.

ctx: CanvasRenderingContext2D

The canvas rendering context.

https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D-+

Canvas functions

clear(x?: number, y?: number, width?: number, height?: number)

Clears the canvas.

fill(style: Style, x?: number, y?: number, width?: number, height?: number)

Fills the canvas in given style.

Style = string | CanvasGradient | CanvasPattern

line(x1: number, y1: number, x2: number, y2: number, style?: Style, lineWidth?: number)

Draws a line from (x1, y1) to (x2, y2) in chosen style and lineWidth.

(Style = string | CanvasGradient | CanvasPattern)

circle(x: number, y: number, radius: number, style?: Style, lineWidth?: number)

Draws a circle with center in (x, y) and given radius in chosen style and lineWidth.

(Style = string | CanvasGradient | CanvasPattern)

rectangle(x: number, y: number, width: number, height: number, style?: Style, lineWidth?: number)

Draws a rectangle with upper left corner in (x, y), given width and height, in chosen style and lineWidth.

(Style = string | CanvasGradient | CanvasPattern)

triangle(x1: number, y1: number, x2: number, y2: number, x3: number, y3: number, style?: Style, lineWidth?: number)

Draws a triangle with corners in (x1, y1), (x2, y2) and (x3, y3) in chosen style and lineWidth.

(Style = string | CanvasGradient | CanvasPattern)

text(value: any, x?: number, y?: number, font?: number | string, style?: Style, lineWidth?: number)

Draws the value, converted to a string, with lower left corner in (x1, y1) in chosen font, style and lineWidth.

(Style = string | CanvasGradient | CanvasPattern)

image(path: string, x?: number, y?: number, width?: number, height?: number): Promise<void>

Draws the image in given path with upper left corner in (x, y).

pixel(x: number, y: number, color?: string)
