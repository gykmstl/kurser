Namnge mapparna text-1, text-2, och så vidare.

Inmatningar från användaren, input, har här markerats med fet stil för att förtydliga att det är inmatningar. Så behöver det inte att se ut i dina program. 

  1. Skapa ett hälsande program som fungerar så här:
    Vad heter du?
    Mattias 
    Hej Mattias!

    Tips: Använd strängaddition "Hej " + namn + "!", eller en strängmall `Hej ${namn}!` vid utskrift.
     
    Start  Test-1  Test-2
     
  2. Detta program ska presentera sin användare på känt manér.
    First Name?
    Last Name?
    James
    Bond
    My name is Bond, James Bond.
     
    Start  Test-1  Test-2

  3. Skriv ett program som skriver ut en inmatad text först enbart med versaler sedan enbart med gemener:
    En text:
    Hej Monika
    Versalt: HEJ MONIKA
    Gement: hej monika

    Tips: Använd strängmetoderna toUpperCase och toLowerCase.

    Start  Test-1  Test-2

  4. Skriv ett program som frågar efter ditt namn varefter din signatur skrivs ut:
    Förnamn?
    Efternamn?
    Kalle
    Anka
    Dina initialer är KA.

    Tips: Använd indexering []. Observera att det första tecknet har index 0 (noll).

    Start  Test-1  Test-2

  5. Skriv ett program som tar ditt namn varefter ditt användarnamn (tre första ur förnamnet + tre första ur efternamnet) skrivs ut: 
    Förnamn?
    Efternamn?
    Kalle 
    Anka 
    Ditt användarnamn är kalank.

    Tips: Använd metoderna slice och slice och toLowerCase. 

    Start  Test-1  Test-2

  6. Skriv ett program som frågar efter ditt telefonnummer (inklusive riktnummer). Byt ut bindestrecket mot mellanslag:
    Telefonnummer med bindestreck:
    0910-12345
    Telefonnummer utan bindestreck: 0910 12345

    Tips: Använd metoden replaceAll.

    Start  Test-1  Test-2

  7. * ( Extrauppgift)
    Skriv ett program som delar upp ett telefonnumret i riktnummer och abonnentnummer på detta vis:
    Telefonnummer med bindestreck:
    0910-12345
    Riktnummer: 0910
    Abonnentnummer: 12345

    Tips: Använd metoden indexOf för att hitta positionen för bindestrecket. Då man vet positionen kan man sedan dela upp numret i två delsträngar.

    Start  Test-1  Test-2

  8. * (Extrauppgift)
    Skriv ett program som använder några ytterligare strängmetoder. Exempelvis så här:
    En text:
    Hello World!
    Innehåller (minst) ett mellanslag: true
    Inleds med "Hej": false

    Start  Test-1  Test-2
